git init Inicia um repositorio Git local
git status Mostra os arquivos modificados
git add nome-arquivo Adiciona um arquivo ao staging
git add. Adicione todos os arquivos
git commit -m "mensagem" Cria um commit com uma mensagem
git log Mostrra o hitórico de commits
git clone <url> Clona um repositório remoto
git push Envia os commits locais para o GitHub (ou outro remoto)
git pull Atualiza o repositório local com base no remoto
